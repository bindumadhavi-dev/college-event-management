# College Event Management System

## Project Description
A web-based application for managing college events, registrations, ticket booking, attendance, and payments.

## Features
- User Registration
- Login System
- Event Management
- Ticket Booking
- Attendance Tracking
- Payment Module
- Admin Panel

## Technologies Used
- HTML
- CSS
- JavaScript
- backend with my sql and python
- mangodb
